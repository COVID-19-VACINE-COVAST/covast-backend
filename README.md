# covast-backend
covast backend repo (django, graphql)

# Docker

## Prerequisites

- Docker
- Docker Compose

## Build
### Docker
```sh
docker build .
```

### Docker Compose
```sh
docker-compose build
```

## Using
### Run Server
```sh
docker-compose up -d
```

### Stop Server
```sh
docker-compose stop
```

### List Containers
```sh
docker ps
```
